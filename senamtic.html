What is Semantics tags in HTML?

Semantic HTML elements are those that clearly describe their meaning to a human and machine in a readable way. Elements such as < header>, < footer> and < article> are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them.

List of new semantic elements

The semantic elements added in HTML5 are:

  • < article>
  • < aside>
  • < details>
  • < figcaption>
  • < figure>
  • < footer>
  • < header>
  • < main>
  • < mark>
  • < nav>
  • < section>
  • < summery>
  • < time>
